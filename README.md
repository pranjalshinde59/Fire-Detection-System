## Fire Detection System using Arduino

## Project Overview
The Fire Detection System is an embedded systems project designed to detect fire and smoke at an early stage using flame and gas sensors. The system processes sensor data through an Arduino microcontroller and immediately activates visual and audible alarms, providing a simple and cost-effective fire safety solution.

## Objectives
- Detect fire and smoke in real time.
- Alert users using a buzzer and LED.
- Demonstrate sensor interfacing with Arduino.
- Develop a reliable embedded safety system.

## Technologies Used
- Arduino Uno
- Embedded C (Arduino IDE)
- Flame Sensor
- MQ-2 Smoke Sensor
- Buzzer
- LED
- Breadboard
- Jumper Wires

## System Workflow
1. Continuously monitor flame and smoke sensor readings.
2. Compare sensor values with predefined thresholds.
3. Detect the presence of fire or smoke.
4. Trigger the buzzer and LED alarm.
5. Continue monitoring until the threat is removed.
